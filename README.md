# HyperJs-Pro-Setup
Configuration to charge your hyperjs terminal into beast.

### Plugins Installed 

```
plugins: [
    "hyperline",
    "hyperlinks",
    "hyper-pane",
    "hyper-simple-highlight-active-session",
    "hyper-snazzy", 
 ],
```

### Requirements

- Hyperjs
- Menlo Font 

### Installation
- Simply copy/move ```.hyper.js``` to your home directory ```~/.hyper.js```
- !!! Enjoy !!!
